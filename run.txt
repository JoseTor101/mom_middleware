Run the following command to generate Python gRPC code:
    python -m grpc_tools.protoc -I. --python_out=. --grpc_python_out=. mom.proto

Run Redis server
    redis-server --daemonize yes

Run MOM server
    python mom_server.py

Run REST API server
    python rest_api.py

Create topic
    curl -X POST "http://localhost:8000/topic/topic_name"

See existing topics
    redis-cli SMEMBERS topics

See topics with pending messages
    redis-cli
    KEYS *
